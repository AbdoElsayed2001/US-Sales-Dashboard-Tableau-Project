# 📊 US Sales Dashboard | Tableau Project


## 🚀 Project Overview

This project is an interactive **Tableau Dashboard** built to analyze U.S. sales data and extract meaningful business insights.

The goal is to transform raw data into clear visual insights that support **data-driven decision making**.
